todoApp/README.md# Todo App

A simple and responsive To-Do application built with vanilla HTML, CSS, and JavaScript.

## Features

- Add new tasks to your to-do list
- Mark tasks as complete with a checkbox
- Delete tasks from the list
- Filter tasks by status (All, Active, Completed)
- Responsive design that works on all devices
- Local storage support for persistent data
- Real-time task updates

## Technologies Used

- HTML5 - Semantic markup
- CSS3 - Modern styling with flexbox
- JavaScript (Vanilla) - DOM manipulation and event handling

## How to Run

1. Open `index.html` in your web browser
2. Start adding tasks to your to-do list
3. Click on a task to mark it as complete
4. Use the delete button to remove tasks
5. Filter your tasks using the filter buttons (All, Active, Completed)

## Project Structure

- `index.html` - Main HTML file with all markup
- `style.css` - CSS styling and layout
- `index.js` - JavaScript logic and event handling

## Usage

1. Enter a task description in the input field
2. Click "Add" button or press Enter to add the task
3. Click the checkbox to mark tasks as complete
4. Click the delete icon to remove a task
5. Use filter buttons to show All, Active, or Completed tasks

## Key Features

- Clean and intuitive user interface
- Real-time task updates without page refresh
- Persistent storage using browser's localStorage
- Smooth transitions and animations
- Mobile-friendly responsive layout
