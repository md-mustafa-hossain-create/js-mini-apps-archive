# Book Tracker

A beginner web app for tracking books.

## Features

- Add new books with title and author
- View your book collection
- Track reading progress

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

1. Open `index.html` in your web browser
2. Use the form to add new books
3. Your books will be displayed in the tracker

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - Styling for the application
- `objectConstructors.js` - JavaScript logic for book management

## How to Use

1. Enter a book title and author in the input form
2. Click the submit button to add the book
3. Your books will appear in the list below
4. Click to manage or view your collection
