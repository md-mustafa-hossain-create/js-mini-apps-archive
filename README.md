## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari).
- A code editor (VS Code recommended).
- Git installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/md-mustafa-hossain-create/js-mini-apps-archive.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd js-mini-apps-archive
   ```

3. **Run a project (e.g., Currency Converter):**
   You can open `currencyConverter/index.html` directly in your browser, or use a local server for better performance:
   ```bash
   # Using Python
   cd currencyConverter
   python -m http.server 8000

   # Or using Node.js http-server
   npx http-server ./currencyConverter
   ```

## Code Quality Standards

We adhere to strict coding standards to ensure readability and maintainability:
- **ES6+ Syntax**: Usage of `const`/`let`, arrow functions, and modules.
- **DRY Principle**: Reusable functions and modular code.
- **Semantic HTML**: Accessible and meaningful markup.
- **Responsive Design**: Mobile-first approach where applicable.
- **Error Handling**: Graceful degradation and user feedback.
